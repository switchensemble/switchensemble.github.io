DECAPP CMS + CLOUDFLARE WORKER OAUTH PROXY TECHNICAL DEBUG HANDOFF
DOCUMENT ==================================

GOAL

Enable non-technical contributors to publish to:

switchensemble/switchensemble.github.io

using: - Decap CMS - GitHub backend - Custom OAuth proxy via Cloudflare
Worker - Hosted at https://www.switchensemble.com/admin/#/

  ------------------------
  1) SYSTEM ARCHITECTURE
  ------------------------

Browser (Decap CMS) │ │ GET /auth ▼ Cloudflare Worker (OAuth Proxy) │ │
302 redirect to ▼ GitHub OAuth authorize │ │ 302 redirect with ?code &
state ▼ Cloudflare Worker (/callback) │ │ POST exchange code →
access_token │ │ Return HTML: │
window.opener.postMessage(“authorization:github:success:TOKEN”) ▼
Browser (Decap) │ │ Uses TOKEN to call GitHub API ▼ GitHub REST API

  -----------------
  2) DOMAIN SETUP
  -----------------

Primary: https://www.switchensemble.com

Non-www: https://switchensemble.com → 301 redirect to www (Cloudflare
rule)

Worker routes: www.switchensemble.com/auth
www.switchensemble.com/callback www.switchensemble.com/health*

  -------------------------
  3) WORKER ENV VARIABLES
  -------------------------

CMS_ORIGIN = https://www.switchensemble.com CALLBACK_URL =
https://www.switchensemble.com/callback GITHUB_CLIENT_ID = (set)
GITHUB_CLIENT_SECRET = (set)
